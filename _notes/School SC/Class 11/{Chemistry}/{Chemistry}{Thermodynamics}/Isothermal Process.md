# Isothermal

- Graph
	- ![[Zoom_KMBMtJreHW.png]]
- Temperature is constant
- $\Delta$T = 0
- If $\Delta$T = 0, then $\Delta$U = 0
	- 0 = q +w
	- q = - w
- If there is no change in temperature (isothermal) there is no change in internal energy
-  W = - P$_{external}$$\Delta$V
